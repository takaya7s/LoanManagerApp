# LoanManagerApp 手動テスト

上から順に実施し、問題なければ `[ ]` を `[x]` に変更してください。

---

## 1. 初回起動

1. `Config`、`Data`、`Logs` フォルダを削除する
2. `LoanManagerApp.exe` を起動する

- [ ] エラーなく起動する
- [ ] exeと同じ場所に `Config`、`Data`、`Logs` が作成される
- [ ] `Config/settings.json` が作成される
- [ ] `Data/loan.db` が作成される
- [ ] `Logs/yyyyMMdd_HHmmss.log` が作成される

---

## 2. 新規登録画面の初期値

「新規登録」を押して確認します。

- [ ] ローン名称が `無題`
- [ ] 借入額が `1,000,000`
- [ ] 年間金利が `15`
- [ ] 返済方式が `元利均等返済`
- [ ] 利息計算が `月割り`
- [ ] 返済条件が `返済期間で設定`
- [ ] ボーナス払いが `なし`

---

## 3. 借入額入力

1. 借入額欄へ `1234567` と入力する
2. 別の入力欄をクリックする
3. 再度、借入額欄をクリックする

- [ ] 入力中は `1234567`
- [ ] フォーカスが外れると `1,234,567`
- [ ] 再度フォーカスするとカンマが消える
- [ ] 文字が右寄せになっている
- [ ] 数字以外の通常文字を入力できない

---

## 4. 入力範囲

- [ ] 借入額 `999円` は登録できない
- [ ] 借入額 `1,000円` は登録できる
- [ ] 借入額 `1,000,000,000円` は登録できる
- [ ] 借入額 `1,000,000,001円` は登録できない
- [ ] 年利 `0%` は登録できる
- [ ] 年利 `99.99%` は登録できる
- [ ] 年利 `100%` は登録できない
- [ ] 初回返済日が借入日以前の場合は登録できない
- [ ] 返済期間は1～600か月だけ登録できる

## 10. ボーナス払い

### UI

- [ ] 初期値が「なし」
- [ ] 「なし」「年1回」「年2回」を選択できる
- [ ] 年1回では月を1つ指定できる
- [ ] 年2回では月を2つ指定できる
- [ ] 年2回で同じ月を指定すると登録できない
- [ ] UIが見切れない

### 年1回

```text

年利: 0%
返済方式: 元金均等返済

```


- [ ] 6月と12月に200,000円ずつ追加される

- [ ] 最終残高が0円

## 11. ローン一覧の並び順

次の順番で登録します。

| 登録順 | 名称 | 借入日 | 借入額 | 残回数 |
|---:|---|---|---:|---:|
| 1 | A | 2026/04/01 | 2,000,000円 | 12回 |
| 2 | B | 2026/04/01 | 1,000,000円 | 24回 |
| 3 | C | 2026/04/01 | 1,000,000円 | 12回 |
| 4 | D | 2026/04/01 | 1,000,000円 | 12回 |
| 5 | E | 2026/03/01 | 500,000円 | 12回 |
| 6 | F | 2026/01/01 | 100,000円 | 完済 |

**期待順**

```text
E → A → B → C → D → F
```

- [ ] 借入日の昇順
- [ ] 同じ借入日は借入額の降順
- [ ] 借入額も同じ場合は残回数の降順
- [ ] 残回数も同じ場合は登録順
- [ ] 完済済みは最下部
- [ ] 完済済みの背景が薄灰色

## 15. 編集・削除・再起動
